# Contributor Covenant 行動規範

## 私たちの約束

メンバー、貢献者、およびリーダーとして、私たちは、年齢、体格、目に見える障害や見えない障害、民族性、性別に関する特徴、性自認および性表現、経験の程度、教育、社会経済的地位、国籍、容姿、人種、カースト、肌の色、宗教、あるいは性的指向にかかわらず、私たちのコミュニティへの参加をハラスメントのない経験とすることを約束します。

私たちは、オープンで、歓迎的で、多様性を尊重し、インクルーシブで健全なコミュニティの実現に貢献する方法で行動し、相互に関わり合うことを約束します。

## 私たちの標準

コミュニティにとって良い環境作りに貢献する行動には、以下のようなものがあります。

- 他者に共感と優しさを示すこと
- 異なる意見、視点、経験に敬意を払うこと
- 建設的なフィードバックをし、また受け入れること
- 自分の過ちの責任を受け入れ、影響を受けた人に謝罪し、その経験から学ぶこと
- 個人だけでなく、コミュニティ全体にとって何が最善であるかに焦点を合わせること

容認できない行動の例には、以下のようなものがあります。

- 性的な言葉や画像の使用、およびあらゆる種類の性的な注目や誘惑
- トローリング、侮辱的あるいは中傷的なコメント、個人的あるいは政治的な攻撃
- 公的もしくは私的なハラスメント
- 明示的な許可なく、物理的または電子メールアドレスなどの他者の個人情報を公開すること
- その他、職業上の場において合理的に不適切とみなされる行為

## 実施の責任

コミュニティのリーダーは、容認できる行動の基準を明確にし、それを実施する責任があり、不適切、脅迫的、攻撃的、または有害とみなされるあらゆる行動に対して、適切で公平な是正措置を講じます。

コミュニティのリーダーは、この行動規範に沿わないコメント、コミット、コード、Wiki の編集、Issue、その他の貢献を削除、編集、または拒否する権利と責任を有し、また、適切と判断された場合には、モデレーションの判断理由を共有します。

## 適用範囲

この行動規範は、コミュニティのすべての空間に適用され、また、個人が公の場でコミュニティを代表している場合にも適用されます。私たちのコミュニティを代表する例には、公式のメールアドレスの使用、公式のソーシャルメディアアカウントを介した投稿、またはオンラインあるいはオフラインのイベントで任命された代表者としての行動などが含まれます。

## 執行

虐待的、嫌がらせ的、またはその他容認できない行為の事例は、GitHub の [Private vulnerability reporting](https://github.com/Mink16/xserver-mcp/security/advisories/new) または GitHub 上で [@Mink16](https://github.com/Mink16) への直接連絡により、コミュニティのリーダーに報告することができます。すべての苦情は、迅速かつ公正に確認および調査されます。

コミュニティのリーダーはすべて、通報者のプライバシーと安全を尊重する義務があります。

## 執行のガイドライン

コミュニティのリーダーは、この行動規範に違反したとみなす行動に対し、結果を判断するために、以下のコミュニティへの影響に関するガイドラインに従います。

### 1. 是正

**コミュニティへの影響**: 不適切な言葉遣い、またはその他コミュニティ内でプロフェッショナルでない、あるいは歓迎されないとみなされた行動。

**結果**: コミュニティのリーダーから、非公開の書面による警告が通達され、なぜその行動が不適切であったかを明確にし、行動の説明が行われます。公の場での謝罪を求められる場合があります。

### 2. 警告

**コミュニティへの影響**: 単一の事件または一連の行動を介した違反。

**結果**: 継続的な行動に対する結果を伴う警告。指定された期間、行動規範を執行するメンバーとの交流を含め、関係する人々との交流を避けること。これには、コミュニティの空間や、ソーシャルメディアなどの外部のチャンネルでの交流を避けることが含まれます。これらの条件に違反した場合、一時的または永続的な追放につながる可能性があります。

### 3. 一時的な追放

**コミュニティへの影響**: 持続的な不適切な行動を含む、コミュニティの標準に対する重大な違反。

**結果**: 一定期間、コミュニティとのあらゆる種類の交流や公のコミュニケーションを一時的に追放します。この期間中、行動規範を執行するメンバーとの交流を含め、関係する人々との公的または私的な交流は許可されません。これらの条件に違反した場合、永続的な追放につながる可能性があります。

### 4. 永続的な追放

**コミュニティへの影響**: 持続的な不適切な行動、個人へのハラスメント、または個人の集団に対する攻撃または侮辱を含む、コミュニティの標準に対する違反のパターンの実証。

**結果**: コミュニティ内でのあらゆる種類の公の交流からの永続的な追放。
